Requirements:

1. Microsoft Visual Studio (Latest)
2. Python 3.8.3
3. OpenCV Version 4.5.2
4. Testing videos (optional)

How to run:

Open the command prompt and go to the root folder of the application. Type:

a) python training_script.py

If you want to run the training script.

NOTE: Running this will replace all the data in NumPy (.npy) files and the main face mask detection script might not work. If this happens, please re-download the .npy files from Github.

b) python mask_detection.py

This will run the main face mask detection script.

NOTE: If you chose to download the testing videos, please make sure they are in the "Testing" folder inside the main folder of this application.
